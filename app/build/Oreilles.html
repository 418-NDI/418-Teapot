Oreilles

Vous êtes dans les oreilles.
Les oreilles humaines captent les sons, permettant de percevoir le monde environnant.
Les océans, quant à eux, sont un immense résonateur. Ils transmettent les sons sur des milliers de kilomètres, permettant à des espèces comme les baleines de communiquer sur de grandes distances.

Le sachiez-vous ?

Les sons voyagent environ 4,3 fois plus vite dans l'eau que dans l'air, une propriété essentielle pour la vie marine.

Les océans permettent une communication longue distance entre les espèces marines, ce qui est crucial pour leur survie.
